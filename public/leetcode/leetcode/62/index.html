62. 不同路径

原题链接

一开始是想用数学的思路解题，奈何数学太差。。。最后换思路用动态规划（入参m,n足够小）：

  • dp[i][j]表示从点0,0到点i,j的所有可能路径数
  • 初始化dp[0][0~n-1]与dp[0~m-1][0]
  • 循环计算其他点，dp[i][j] = dp[i-1][j] + dp[i][j-1]
class Solution {
public:
    int uniquePaths(int m, int n) {
        int dp[100][100];
        memset(dp, 0, sizeof(int)*100*100);

        dp[0][0] = 1;
        for (int i = 0; i < m; ++i)
            dp[i][0] = 1;
        for (int j = 0; j < n; ++j)
            dp[0][j] = 1;
        
        for (int i = 1; i < m; ++i)
            for (int j = 1; j < n; ++j)
                dp[i][j] = dp[i-1][j] + dp[i][j-1];

        return dp[m-1][n-1];
    }
};

官方题解中有数学解法的思路：

从左上角到右下角的过程中，我们需要移动m+n-2次，其中有m-1次向下移动，n-1次向右移动。 因此路径的总数，就等于从m+n-2次移动中选择m-1次向下移动的方案数字，即组合数：

$$ C_{m+n-2}^{m-1} = \tbinom{m-1}{m+n-2} = \frac{(m+n-2)!}{(n-1)!\cdot(m-1)!} = \frac{(m+n-2)\cdot(m+n-3)\cdots n}{(m-1)!} $$

class Solution {
public:
    int uniquePaths(int m, int n) {
        long long ans = 1;
        for (int x = n, y = 1; y < m; x++,y++) {
            ans = ans*x/y;
        }
        return ans;
    }
};
